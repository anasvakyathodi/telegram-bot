<h2>🛠️ Installation Steps:</h2>

<p>1. Setting up Environment</p>

```
virtualenv env && source env/bin/activate
```

<p>2. Installing dependencies</p>

```
pip install -r requirements.txt
```

<p>3. Running Server</p>

```
python server.py
```
